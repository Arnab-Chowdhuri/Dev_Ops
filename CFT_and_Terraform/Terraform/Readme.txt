Install Terraform in windows : 
go to terraform.io -> Download Terraform -> select windows from left pannel -> download Windows (AMD 64) -> Make Terraform folder in C:\Terraform and keep the downloaded zip folder inside that and ectract there -> Now add the path of this file to environment variables, search for "Edit the system Environment Variablles" -> Under System Variables select path and click edit, now click new and copy paste the path of that extracted file here and do OK Ok OK.

Come in CMD prompt or VS Code :
do -> <terraform -v> to see the version
in Vs code go to extention and install Hashicorp Terraform -> restart the Vs Code -> open the terminal and check by doing -> <terraform -v>
